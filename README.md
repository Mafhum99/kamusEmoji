# 🎨 Kamus Emoji Premium

> **Cari, Temukan, dan Salin Emoji Favorit Anda dengan Mudah!**

Kamus Emoji Premium adalah aplikasi web modern dan responsif yang membantu Anda menemukan emoji sempurna untuk setiap kesempatan. Dengan antarmuka yang elegan dan fitur pencarian yang cepat, menyalin emoji favorit Anda belum pernah semudah ini!

---

## ✨ Fitur Utama

- 🔍 **Pencarian Cepat** - Cari emoji berdasarkan nama, kata kunci, atau kode Unicode
- 🏷️ **Filter Kategori** - Browse emoji berdasarkan kategori (Wajah, Tangan, Hati, Hewan, Makanan, dll)
- 📋 **Salin Satu Klik** - Klik emoji untuk langsung menyalin ke clipboard
- 🎨 **Desain Modern** - UI/UX yang cantik dengan tema gelap dan efek glassmorphism
- 📱 **Responsif** - Bekerja sempurna di desktop, tablet, dan mobile
- ⚡ **Performa Tinggi** - Optimized untuk ribuan emoji
- 🌈 **Emoji Lengkap** - Ribuan emoji dari berbagai kategori

---

## 🚀 Cara Menggunakan

### 1. Buka Aplikasi
Buka file `index.html` di browser Anda (Chrome, Firefox, Edge, Safari, dll)

### 2. Cari Emoji
- Ketik di kolom pencarian untuk mencari emoji
- Contoh: `smile`, `heart`, `U+1F600`, atau bahkan `🔥`

### 3. Filter Kategori
Klik tombol filter di bawah search box untuk memfilter berdasarkan kategori:
- 😊 **Wajah** - Emoji ekspresi wajah
- ✋ **Tangan** - Gestur tangan
- ❤️ **Hati** - Berbagai jenis hati
- 🐶 **Hewan** - Emoji hewan
- 🍕 **Makanan** - Emoji makanan & minuman
- ✈️ **Travel** - Emoji perjalanan & tempat
- 📦 **Benda** - Emoji objek
- 🔣 **Simbol** - Simbol & tanda

### 4. Salin Emoji
Klik emoji yang diinginkan untuk menyalin otomatis ke clipboard!

## 🌟 Tips Pro

1. **Gunakan Keywords**: Cari dengan kata kunci seperti `party`, `cool`, `awesome` untuk hasil lebih relevan
2. **Kode Unicode**: Jika tahu kode Unicode (misal: `U+1F600`), langsung cari untuk hasil spesifik
3. **Filter Kombinasi**: Gunakan search + filter kategori untuk hasil lebih spesifik
4. **Mobile Friendly**: Gunakan di mobile untuk pengalaman terbaik
